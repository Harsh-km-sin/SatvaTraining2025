Understanding target="_top", target="_parent", and target="_self"

This page demonstrates the difference between the three target attributes:

Links with different target attributes

Target="_self": Opens the link in the same window/tab (default behavior).

Open Example in the Same Tab

Target="_parent": Opens the link in the parent frame (if in an iframe).

Open Example in Parent

Target="_top": Opens the link in the full body of the window, removing any frames.
